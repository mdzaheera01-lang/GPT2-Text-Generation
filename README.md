# GPT-2 Text Generation

This project demonstrates text generation using GPT-2 and the Hugging Face Transformers library.

## Features
- Generates text from user prompts
- Uses pre-trained GPT-2 model
- Interactive terminal input

## Technologies Used
- Python
- GPT-2
- Hugging Face Transformers

Machine Learning

## Outcome
Successfully generated coherent text using a pre-trained GPT-2 model.
